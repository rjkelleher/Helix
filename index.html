About this data

The interface — sorting, filtering, charts, the stock detail view — is fully real. Only the numbers are illustrative until you connect a feed.

* next to a ticker marks a current S&P Biotech (XBI) constituent. In demo mode only the confirmed top names are flagged; when you build live data, the full, current XBI membership is loaded automatically from State Street's daily holdings file. Use the XBI filter to show only constituents.

To load your real numbers: run the included build_dashboard_data.py against tickers.csv. It writes a data.json next to this page; reload and the badge turns green with your live snapshot.

Connect a live feed

Two ways to get real data into HELIX:

1 · Snapshot (recommended). Run build_dashboard_data.py; it produces data.json (prices, history, fundamentals, news) for every ticker in tickers.csv. This page loads it automatically.

2 · Live quotes & news in the detail view via a free Finnhub key (optional; US names most reliable):

Stored only in this browser. Leave blank to stay on snapshot/demo data.

With a key connected, a Live column appears in the table (US names only): it shows the pre-market move before the open and the live intraday change once the market opens. It fetches the rows on screen, so it fills in as you scroll. Pre-market prints for thinly-traded small caps may be missing, and true pre-market data depends on your Finnhub tier.
